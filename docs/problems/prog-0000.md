## คำอธิบายโจทย์

!!! note ""
    โจทย์ให้นำเข้าจำนวนเต็ม $a$ และ $b$ แล้วส่งออกผลรวมของ $a$ และ $b$

## เฉลย

??? note "เฉลย"
    เราสามารถรับค่า a และ b แล้วนำมาบวกกันตรงๆ ได้เลย
    ??? note "โค้ด"
        === "C"
            ```c
            #include <stdio.h>

            int main() {
                int a, b;
                scanf("%d %d", &a, &b);
                printf("%d", a+b);
            }
            ```
        === "C++"
            ```cpp
            #include <iostream>

            using namespace std;

            int main() {
                int a, b;
                cin >> a >> b;
                cout << a+b;
            }
            ```
